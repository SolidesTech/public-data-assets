# Documentação dos Conjuntos de Dados

Este diretório contém a documentação detalhada de cada conjunto de dados disponível no repositório.

## Estrutura de Cada Documento

Cada arquivo de documentação (`<nome-do-dataset>.md`) deve conter:

- **Descrição**: O que são esses dados e para que servem.
- **Fonte**: Origem dos dados e URL de referência.
- **Licença**: Licença de uso dos dados originais.
- **Atualização**: Frequência com que os dados são atualizados.
- **Dicionário de Variáveis**: Descrição de cada coluna/campo.
- **Notas**: Observações relevantes sobre qualidade, limitações ou peculiaridades.

# public-data-assets

Repositório para ativos de dados públicos da [SolidesTech](https://github.com/SolidesTech).

## Visão Geral

Este repositório centraliza ativos de dados públicos utilizados em projetos da SolidesTech. Os dados aqui disponibilizados são de uso público, organizados e documentados para facilitar o acesso, a integração e a reprodutibilidade.

## Estrutura do Repositório

```
public-data-assets/
├── data/               # Conjuntos de dados públicos
│   ├── raw/            # Dados brutos, sem processamento
│   └── processed/      # Dados processados e prontos para uso
├── docs/               # Documentação dos conjuntos de dados
├── scripts/            # Scripts de coleta, limpeza e transformação de dados
└── README.md           # Este arquivo
```

## Como Usar

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/SolidesTech/public-data-assets.git
   ```

2. **Explorar os dados:**
   Acesse a pasta `data/` para encontrar os conjuntos de dados disponíveis.
   Consulte a pasta `docs/` para a documentação de cada conjunto.

3. **Executar scripts:**
   Os scripts na pasta `scripts/` permitem baixar, limpar e transformar os dados.
   ```bash
   python scripts/<nome-do-script>.py
   ```
